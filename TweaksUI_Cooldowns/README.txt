TweaksUI: Cooldowns
===================
Version 2.0.0

Requires World of Warcraft 12.0.0 (Midnight) or later.
For The War Within, use version 1.5.x.

Formerly known as Cooldown Manager Tweaks (CMT)

DESCRIPTION
-----------
TweaksUI: Cooldowns enhances Blizzard's Cooldown Manager system with:
- Custom grid layouts for all cooldown trackers
- Per-tracker appearance settings (size, spacing, aspect ratio, zoom)
- Visibility controls based on combat, group, target, and instance type
- Custom tracker for spells, items, and equipped gear
- Persistent buff display with inactive state styling
- Proc and buff alert system (in development)

INSTALLATION
------------
1. Extract the TweaksUI_Cooldowns folder to your Interface/AddOns directory
2. Launch WoW and enable TweaksUI: Cooldowns in the addon menu

NOTE FOR CMT USERS
------------------
If you previously used Cooldown Manager Tweaks, your settings will be
automatically migrated when you first load TweaksUI: Cooldowns.

COMMANDS
--------
/tuicd           - Open settings hub
/tuicd debug     - Toggle debug mode  
/tuicd reset     - Reset all settings
/tuicd resetpos  - Reset tracker positions
/tuicd help      - Show all commands
/tuicdlayout     - Toggle layout mode (drag trackers)
